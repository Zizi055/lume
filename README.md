# LUMÉ — сайт стоматологической клиники

Современная стоматология полного цикла для взрослых, детей и семей. 

## Стек

- HTML / CSS / vanilla JS
- IntersectionObserver — плавное появление секций при скролле
- CSS `:has()` — hover-превью в списке услуг без JS
- Google Fonts: Manrope + Instrument Serif
- Фотографии — Unsplash, свободная лицензия (Unsplash License)

## Структура

```
index.html      — главная страница
css/style.css   — стили
js/main.js      — меню, скролл-состояние хедера, reveal-анимации, параллакс, форма записи
```
